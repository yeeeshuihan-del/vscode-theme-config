### 🧛‍♂️ VS Code 配置：基于 Dracula 主题

这是一个基于 **Dracula 主题** 深度定制的 VS Code 配置方案，涵盖了 **界面外观** 与 **编辑器功能** 的全方位优化。

#### 🛠️ 前置条件

在应用以下配置前，请确保已安装 Dracula 主题：

1. 进入菜单栏 **Settings** > **Theme** > **Color Theme**。
2. 选择 **Browse Additional Color Themes...**。
3. 在列表中找到并安装 **"Dracula Theme"**。

#### ⚙️ 如何应用配置

1. 打开 VS Code 顶部命令面板（快捷键 `Ctrl+Shift+P` 或 `F1`）。
2. 输入并选择以下命令：
   `> Preferences: Open User Settings (JSON)`
3. 将settings.json的代码内容完整复制并粘贴到你的 `settings.json` 文件中，保存即可生效。
